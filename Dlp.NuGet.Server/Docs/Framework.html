What is DLP Framework

DLP Framework is an assembly with a set of services and methods elaborated to simplify and standardize simple everyday tasks. The methods provided by the DLP Framework were tested and approved, being used in real production applications, ensuring quality and safety. In addition to this, all the code possess 100% code coverage by unit tests, giving you the security to concentrate on your application and focus exclusivelly on your business logic.

The framework is primally consisted of extension methods, creating a transparent flow in the development process, without the need to instantiate several classes just for simple operations.
